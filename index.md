# Welcome to My GitHub Pages

## Introduction

This is the introduction to my GitHub Pages site. Here, I will be sharing my data science tutorial on analyzing the housing crisis using the Federal Housing Finance Agency (FHFA) House Price Index dataset.

## About the Tutorial

In this tutorial, we will walk through the entire data science pipeline, including:

- Data curation and preprocessing
- Exploratory data analysis
- Hypothesis testing and machine learning
- Generating insights and conclusions

We will be using the FHFA House Price Index dataset to examine the impact of the housing crisis on the U.S. housing market. The dataset provides comprehensive data on house price changes at the national, state, and metropolitan levels.
